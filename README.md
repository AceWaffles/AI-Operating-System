# AI Operating System (AIOS)

Purpose:

Create a shared institutional memory between Michael and AI participants.

Objectives:

- Synchronize projects
- Preserve decisions
- Reduce context switching
- Prevent idea loss
- Prioritize work

Core Concepts:

Workspace
↓
Morning Meeting
↓
Historian
↓
Repository
↓
Institutional Memory

AI Participants:

- Michael (Architect)
- ChatGPT (Strategist)
- Gemma (Engineer)
- Historian AI (Future)
- Chief of Staff (Future)

Rules:

1. Morning Meetings are immutable history.
2. Do not edit old meetings.
3. New information supersedes old information.
4. Ideas are categorized before development.
5. Not every idea becomes work.
